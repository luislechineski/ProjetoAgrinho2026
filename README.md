# ProjetoAgrinho2026
Este site foi desenvolvido com partre do Projeto Agrinho, com objetivo de apresentar a importância e as características do agro no sudoeste do Paraná, destacando a agricultura, a sustentabilidade,a preservação ambiental e as tecnologias ultilizadas no campo de forma criativa e imperativa.

O ptojeto tambem serve como prática de programação web utilizando linguagem HTML, CSS, e JavaScript.

# Funcionalidade
- Layout moderno e resposivo;
- Galeria de imagens do agro;
- Cards informativos;
- Navegação suave;
- Menu de acessibilade;
- Efeitos visuais;
- Quiz interativo.
